# Guess The Word Application

## Name: Untangle

## Key features

- Load the word of the day from API
- Count user guess retries
- Display word definition and examples
- Play word pronunciation audio

## Architecture

- Clean Architecture

## Stack

- kotlin
- Jetpack compose
- Dagger Hilt
- Coroutines
- Retrofit
